In this series, we are going to:
- Setup a [Kubernetes](https://kubernetes.io) cluster
- Deploy [Rabbitmq](https://www.rabbitmq.com) cluster
-  Write, Dockerize and deploy simple python applications as producer/consumer and connect them to rabbitmq.
- Then our [Prometheus](https://prometheus.io) monitoring will be deployed so we can monitor our rabbitmq and use HPA for adding more consumers if queue length gets long.
